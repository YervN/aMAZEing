🖼️ Game Image Placeholder Generator

Click "Download PNG" to save each placeholder, then replace with AI-generated art

How to add real images:
  1. Use an AI image generator (DALL-E, Midjourney, etc.) to create a square thumbnail for each game
  2. Save the image as images/[game-id].png (filenames shown below each card)
  3. In index.html, find the game entry and change image: null to image: "images/[game-id].png"
